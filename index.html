01 · KEY TAKEAWAY

AI produces drafts. It does not make decisions.

Models improved significantly this year. Final judgment remains with the user.

Prompting

Keep prompts concise

AI is a data processing tool. Excessive instructions in a prompt can degrade output quality.

Voice capture

Capture voice data directly

iPhone Notes live transcribe, Otter AI, and Whisper AI convert spoken input to text before drafting begins.

Model selection

Match model to task

Use a lighter model such as Haiku for routine tasks. Reserve a higher tier model such as Sonnet 5 for complex tasks.

02 · MODEL COMPARISON

Claude, Gemini, and ChatGPT: distinct strengths

Summary of comparative observations made during the session.

Claude

  • Assessed as the strongest general model currently available.
  • Fable 5 is restricted for export in the US due to its capability level.
  • Use tiered models matched to task complexity rather than a single model for all tasks.

Gemini

  • Stronger performance for photo and video generation compared to Claude.
  • Canvas function builds a working app, such as a clock, in approximately five minutes.
  • Converts unstructured graphics or charts into structured tables.

ChatGPT

  • Add-in function operates within Excel and PowerPoint for summarising, pivot tables, and formatting tasks.
  • Restricted to personal laptops. Does not operate on managed work devices.

04 · WORKING RULES

Five rules for effective AI use

Output of the in-session exercise: how should a sales team use AI well.

01

Use AI for the first draft only

Let AI structure the initial message. Add the specific detail only the user would know before sending.

02

Run pre-call research through AI every time

This is the highest return use case. Feed it context to surface background and relevant signals before a conversation.

03

Do not use AI for negotiation or closing

Adoption data confirms minimal use at this stage. Trust and judgment do not template.

04

Automate reporting and forecasting

This is the most underused application. Time saved on admin converts directly to time on core tasks.

05

Review all output before sending

AI provides speed, not judgment. Always complete a final read-through before sending.

05 · QUICK REFERENCE

Practical tips

ChatGPT in Office

  • Select Add-in in Excel or PowerPoint to enable ChatGPT within the file.
  • Use for summarising data, building pivot tables, and fixing formatting.
  • Personal laptops only.

Gemini Canvas

  • Canvas builds working apps from a text description.
  • The clock app below was generated in approximately five minutes.
  • Illustrates the speed shift affecting software development timelines.

Voice capture

  • iPhone Notes live transcribe converts speech to text in real time.
  • Otter AI and Whisper AI are alternatives for longer recordings.
  • Feed the transcript to AI for a first structured draft, then edit manually.
